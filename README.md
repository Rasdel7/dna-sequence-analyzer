# DNA Sequence Analyzer 🧬

Bioinformatics tool for analyzing DNA sequences —
GC content, codon translation, ORF detection
and pattern/motif search.

## Live Demo
[Click here](YOUR_STREAMLIT_URL)

## Features
- Nucleotide frequency and pie chart
- Complement and reverse complement strands
- Full codon table (64 codons → amino acids)
- Open Reading Frame (ORF) detection
- DNA motif/pattern search with position map
- Sliding window GC content analysis
- Common restriction sites (EcoRI, BamHI, TATA box)

## Tools Used
- Python, Streamlit, Plotly, Pandas, NumPy

## How to Run Locally
pip install streamlit pandas numpy plotly
streamlit run app.py
